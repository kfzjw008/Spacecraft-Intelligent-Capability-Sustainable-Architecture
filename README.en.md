# 支持自适应科学探测的航天器智能能力可持续构建架构

#### Description
哪吒架构（Spacecraft Intelligent Capability Sustainable Architecture，简称SICSA）首先按照CCSDS SOIS架构，自底向上将底层分散的不同子网，通过汇聚层协议进行归一化设计，使得亚网层和应用支持层这些设备级业务可以进行通用化设计，实现应用层业务完全与底层网络和硬件的脱离。
